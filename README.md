# Portfolio

This is a personal portfolio website showcasing projects, gallery, contact information, and more. Built with HTML, CSS, and JavaScript.

## Features
- Home, About, Projects, Gallery, and Contact pages
- Responsive design
- Animations and interactive elements

## Usage
Open `index.html` in your browser to view the site locally.

## Deployment
You can deploy this site using GitHub Pages or any static hosting provider.
